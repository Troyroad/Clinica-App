# Clínica App — Electron + React + MySQL (Offline)

## Pasos rápidos
1. Instala Node.js y MySQL localmente.
2. Crea la base de datos: `mysql -u root -p < db/schema.sql` (modifica usuario/contraseña según necesites).
3. Ajusta las credenciales de MySQL en `electron-main.js`.
4. Ejecuta:
   ```
   npm install
   npm run dev
   ```
5. La app abrirá una ventana de Electron con el frontend React.
